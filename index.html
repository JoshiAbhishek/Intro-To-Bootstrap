Intro To Boostrap

This is a companion website for an Intro To Bootstrap workshop by Abhishek Joshi. The following code and project stages are for an example website made with Bootstrap for a "Coffee Appreciation Club."

Getting Started

To start working on the Coffee Appreciation Club website, we need a basic HTML website structure. We'll put this in a file called "index.html"

Stage 3

The thirds stage of our website invloves adding the Join section form for new members, and the table of members in the Members section. Some additional JavaScript code is present to add submitted members from the Join from to the Members table.

See Project 
    <!--Join Section-->

    <div id="join" class="jumbotron">
        <div class="container">
            <h2>Join</h2>

            <form id="form">
                <div class="form-group">
                    <label for="firstName">First Name</label>
                    <input type="text" class="form-control" id="firstName" placeholder="Enter Your First Name">
                </div>
                <div class="form-group">
                    <label for="lastName">Last Name</label>
                    <input type="text" class="form-control" id="lastName" placeholder="Enter Your Last Name">
                </div>
                <div class="checkbox">
                    <label>
                        <input id="officer" type="checkbox"> Officer
                    </label>
                </div>
                <button type="submit" class="btn btn-primary">Submit</button>
            </form>
        </div>
    </div>

    <!--Members Section-->
    <div id="members" class="jumbotron">
        <div class="container">
            <h2>Members</h2>
            <table id="members-table" class="table table-hover table-striped">
                <thead>
                    <tr>
                        <th>First Name</th>
                        <th>Last Name</th>
                        <th>Officer</th>
                    </tr>
                </thead>
                <tbody>
                    <tr>
                        <td>Joe</td>
                        <td>Smith</td>
                        <td>Yes</td>
                    </tr>
                    <tr>
                        <td>Mary</td>
                        <td>Gomez</td>
                        <td>Yes</td>
                    </tr>
                </tbody>
            </table>
        </div>
    </div>

    <script>
        $("#form").submit(function(e){
            e.preventDefault();
            $("#members-table > tbody:last-child").append("<tr><td>" + $("#firstName").val() + "</td><td>" + $("#lastName").val() + "</td><td>" + (($("#officer").is(':checked') == true) ? "Yes" : "No") + "</td></tr>");
            $("#firstName").val("");
            $("#lastName").val("");
            $("#officer").attr("checked", false);
    });
    </script>
